# Army Weapons Qualification Tracker

The Army Weapons Qualification Tracker is one segment of an ongoing Leader's Book project designed to support Army leadership in managing Soldier data and readiness. Built with HTML, CSS, and JavaScript, this tracker provides an editable table for text input. There is no calculation provided. This tracker is a simple tool to maintain Soldier data for easy referencing.

The table consists of six columns: Name, Rank, Weapon, Score, Qualification, and Date. Each column is provided with 10 blank rows to provide a team leader with efficient space for annotating a team's full roster. For higher positioned leadership or teams of greater size, the code can be easily adjusted to accommodate by copying and pasting the table row block inside the table body to create the required additional rows.

This project reinforced my understanding of HTML structure, CSS table styling, and GitHub project development. Building this tool also highlighted the importance of maintaining accurate Soldier data to support leadership decisions and contribute to overall unit readiness and promotion eligibility.
