## What is Bootstrap 5?

First, what is Bootstrap 5? According to w3schools.com, Bootstrap is the most popular open-source front-end framework used for “developing creating responsive and mobile-first websites”, with Bootstrap 5 being the newest version. It is basically a toolkit used to help developers create visually appealing websites by providing pre-built components and styles.

## Why not use raw HTML and CSS?

Why not use raw HTML and CSS?
Are there any benefits to using UI (user-interface) frameworks like Bootstrap 5? Why not just use raw HTML and CSS? Well, commonly, GUI (graphical user interfaces) designers created front-end GUI, while programmers created servers that powered the front-end. These days, the skills to work both the front and back-end is becoming more necessary. 

## My Experience

As someone still finding my footing in the coding world, learning web development can sometimes feel overwhelming. There are so many things to remember from writing clean HTML to styling everything just right with CSS. But once I started using UI frameworks like Bootstrap 5, things began to click into place for me. It’s almost like having a helpful guide to get you started and speed up the process a bit.

Bootstrap made it a lot easier for me to understand the structure and flow of building a webpage. I could see how everything fit together without needing to style every little piece from scratch. Instead of spending half my time trying to make a navigation bar look decent, I could just use Bootstrap’s pre-built classes and focus more on the layout or the content I wanted to display. It honestly saved me hours, and for someone who’s still learning.

Another thing I really like about Bootstrap is how it helps me remember the setup of a webpage. After using it for a while, I started to pick up on the patterns rows, columns, containers and how they make a site responsive. Even when I go back to plain HTML and CSS, I find myself thinking in that organized Bootstrap way, which helps me stay more consistent.

Of course, frameworks like Bootstrap aren’t perfect. One issue, which may not be one for someone experience, is that it can sometimes feel like you’re not really learning how to do things from scratch. If you rely too much on it, you might not fully understand how the CSS behind it works. Also, since so many people use Bootstrap, a lot of websites can end up looking kind of similar if you don’t customize the styles. But overall, I think the benefits far outweigh those small issues especially for beginners like me.

In short, UI frameworks make web development feel more accessible and enjoyable. Bootstrap 5 in particular has helped me gain confidence and speed in building websites. It’s a tool that not only saves time but also teaches you structure and best practices along the way. I know that as I keep learning, I’ll eventually dive deeper into custom CSS, but for now, Bootstrap gives me a solid foundation to keep improving.
